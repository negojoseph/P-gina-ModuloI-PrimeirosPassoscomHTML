História dos computadores

Voltar1

Vimos nas aulas que hoje alguns pesquisadores consideram que estamos vivendo na 5a Geração de computadores, tivemos em 1944 o Harvard Mark 1 seguido em 1946 pelo Colossus e ENIAC fechando assim o que foi considerado a primeira geração de computadores.
Com a invenção dos transistores nasceu em 1959 a 2a Geração de computadores.
De 1965 a 1970 tivemos a 3a Geração de computadores, esses por sua vez ja contando com Circuito integrado (transistor, capacitores e resistores). A 4a Geração veio evoluindo desde de 1971 até os dias atuais com os desktops, notebooks e smartphones, com tudo alguns dizem que estamos vivendo a 5a Geração que conta com AI, CQ, CM e metaverso.

O que são Clients?

Voltar2

Clients é o dispositivo ou um usuário que está consumindo alguma coisa da internet/de um servidor.

"É o cliente quem sempre inicia a comunicação com o servidor. O cliente mais conhecido é o navegador. Quando acessamos um site pelo navegador, uma requisição é disparada pela internet para o computador no qual esse site está hospedado"

O que são Servers?

Voltar3

Servers são computadores que armazenam dados, podendo ser local (mais utilizados em empresas), ou via internet (comumente utilizado).

"O papel do servidor web é receber uma requisição e devolver uma resposta para o cliente. Geralmente o servidor não envia dados se o cliente não disparar primeiro uma requisição."

Como é uma estrutura básica de um HTML?

Voltar4

A estrutura básica de um HTML é composta por tags, começando pela tag <html></html> (indicando a linguagem de marcação), dentro dela temos a tag <head></head> (cabeça), e abaixo dela ainda dentro da tag html temos a tag <body></body> (corpo) que vai receber todo o conteúdo que será apresentado na página.

Entendendo a diferença entre listas ordenadas e não ordenadas

Voltar5

Listas Ordenadas (OL): São as listas que como o próprio nome já diz, tem uma ordem lógica a seguir, ela pode ser com números, letras ou algarismos romanos.
Listas não ordenadas (UL): São listas pontuais onde a ordem não importa ou não tem um significado específico, elas podem ser com quadrados ou círculos.
As listas contem dentro delas uma tag específica para o funcionamento das mesmas que é a tag <li>. As tags para fazer listas ordenadas e não ordenadas assim como outras tags você encontra no site: w3c.org
